# 🔍 네이버 SEO 진단 툴 (간단 버전)

**PostCSS/Tailwind 없이 순수 CSS로 만든 깔끔한 버전!**

## ✨ 특징

- ✅ PostCSS 없음 - 빌드 에러 없음!
- ✅ Tailwind 없음 - 간단한 CSS만 사용
- ✅ 모든 기능 동일 - SEO 분석 기능은 완전히 동일
- ✅ 빠른 빌드 - 의존성 최소화
- ✅ Vercel 배포 완벽 지원

## 🚀 빠른 시작

\`\`\`bash
# 설치
npm install

# 개발 서버
npm run dev

# 빌드
npm run build

# 프로덕션 실행
npm start
\`\`\`

## 📦 포함된 기능

- 18개 SEO 진단 항목
- 키워드 연관성 분석 (최대 5개)
- 확장검색 최적화 점수
- 실시간 분석 결과

## 🌐 배포

### Vercel
1. GitHub에 업로드
2. Vercel에서 Import
3. 설정:
   - Framework: Vite
   - Build Command: `npm run build`
   - Output Directory: `dist/public`
   - Install Command: `npm install`
4. Deploy!
